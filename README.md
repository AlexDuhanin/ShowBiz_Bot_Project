Бот для записи клиентов
1. Нажмите старт - бот предложит выбрать направление
2. После выбора направление предложит выбрать свободное время
3. Потом попросит ввести имя и контакт
4. Потом подтвердить запись
5. Админу приходит собщение содержащее время, дату и контакт
6. Также это можно посмотреть в google календаре

Для установки зависимостей введите команду
pip install -r requirements.txt

Команда для запуска бота
python main.py

Для работы бота понадобится прокси одного из двух типов: SOCKS5 или HTTP(S)
